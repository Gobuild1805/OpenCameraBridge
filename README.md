# OpenCameraBridge
## Development status

OpenCameraBridge is currently in early alpha development.

OpenCameraBridge is an open-source application that connects RTSP cameras to cloud services and 3D printer ecosystems.

## Goals

- Generic RTSP support
- ONVIF camera discovery
- Prusa Connect integration
- Tapo camera support
- Reolink support
- Hikvision support
- Dahua support
- Multi-camera management
- Windows, Linux, Docker, and Raspberry Pi support
